# hsimp-cli

CLI for [How Secure Is my Password](https://howsecureismypassword.net/).
Does not send anything over the network, uses HSIMP libraries.

## Installation

`npm i -g hsimp-cli` (or use with `npx`)

## Usage

`hsimp`
